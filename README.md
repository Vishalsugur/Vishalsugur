### 📌 Featured Projects

**[JUMO Laser Trimming — Trustworthy AI System](https://github.com/vishalsugur/jumo-laser-trimming)**
Production-grade ML system predicting the Nominal Correction Value for platinum thin-film temperature sensors using real manufacturing data from JUMO GmbH & Co. KG. Ridge Regression + Random Forest forward model achieving 99.5% tolerance-class yield. Applied VDE SPEC 90012 (VCIO) Trustworthy AI framework with fairness analysis and human-in-the-loop override logging. `Python` `scikit-learn` `MLflow` `DVC` `PostgreSQL` `Power BI`

**[Credit Default Prediction](https://github.com/vishalsugur/credit-default-prediction)**
Compared 4 ML models on 30,000 Taiwanese credit card clients to predict payment default. XGBoost achieved 82.9% accuracy / 0.83 AUC, reducing false negatives by 16% vs. baseline using `scale_pos_weight` for class imbalance. `Python` `XGBoost` `Random Forest` `SVM`

**[Telco Customer Churn Dashboard](https://github.com/vishalsugur/telco-customer-churn-dashboard)**
Interactive R Shiny dashboard analyzing customer churn drivers — dynamic filtering, KPI tracking, and Plotly visualizations. [Live demo →](https://76fyv9-sakshi-bhor.shinyapps.io/project/) `R` `Shiny` `Plotly`

**[AI Ethics & Legal Case Studies](https://github.com/vishalsugur/ai-ethics-case-studies)**
Written analyses of legal and ethical issues in real-world AI deployments, covering GDPR compliance and automated decision-making rights.
